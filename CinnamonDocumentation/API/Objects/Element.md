Stores and displays your UI in space.

Properties:

Enabled : Boolean
- Determines if your UI is being displayed or not
Debug : Boolean
- Adds/Removes the element from the Debug visualizer

Members:

UI : GuiObject
- The UI thats being displayed
Parent2D : GuiObject | ScreenGui
- The UI's parent UI
Instance : Part
- The physical Part that holds your UI
Container: UIContainer | ScreenContainer
- The parent Container of the element
Type : string
- Used for typechecking
